Что такое CSS?

CSS (Cascading Style Sheets) - это язык разметки, который используется для оформления и стилизации веб-страниц. Он позволяет определять внешний вид элементов HTML, таких как цвета, шрифты, отступы, размеры и расположение элементов на странице. CSS позволяет создавать красивый и современный дизайн веб-сайтов и упрощает процесс обслуживания и изменения внешнего вида веб-страниц.

CSS также позволяет создавать анимации, трансформации и эффекты перехода для элементов на веб-странице. Он также поддерживает медиазапросы, позволяющие создавать адаптивные и отзывчивые дизайны, которые могут корректно отображаться на различных устройствах и экранах. CSS используется в сочетании с HTML и JavaScript для создания интерактивных и стильных веб-страниц.

"Примечательный факт, что все веб-браузеры уже имеют свои встроенные стили отображения веб-страниц, но с помощью CSS их можно перезаписать "под себя"."

Кроме того, CSS позволяет создавать различные макеты страниц, управлять расположением элементов, изменять шрифты, цвета, размеры и другие стилевые атрибуты. Он также поддерживает возможность создания анимированных меню, слайдеров, галерей и других интерактивных элементов. CSS имеет множество возможностей для создания красивого и функционального дизайна веб-страниц.

Медиа-запросы

Медиа-запросы в CSS - это база для создания отзывчивой верстки, позволяющая стилизовать элементы в зависимости от размера экрана или устройства, на котором отображается веб-сайт. Технически Медиа-запрос - это простое логическое выражение, которое может быть или истинным или ложным. Условиями для такого выражения являются либо параметры устройства, на котором отображается веб-страница, либо размер экрана в пикселах. В данном руководстве мы рассмотрим базовые возможности медиа-запросов, которые необходимы для адаптивной верстки сайтов и практически полезны. Медиа-запрос пишется в самом файле стилей или в теле документа (тег style) и начинается с объявления правила @media.

Условием может выступать либо устройство - all (все устройства), screen, print, tv и т.д., либо медиа-функции, которые задают параметры устройства или разрешение экрана, на котором отображается документ.

Здесь устройства с максимальным разрешением экрана 480px или минимальным разрешением 320px будут отображать текст тега с классом .my-class серым. Данное условие я привел для примера, практически оно бесполезно. Чаще всего требуется указать либо только максимальное разрешение, либо только минимальное, в пределах которого будет прменяться свойство.

Кроме всего прочего, как мы видим из примера, функции могут содержать условия and (И), not (НЕ) и only (Для старых браузеров, не поддерживающих медиа-запросы). Нет логического оператора or (ИЛИ), его роль выполняет запятая. Медиа-функции, как мы видим заключают в обычные круглые скобки. Размещение свойств в медиа-запросе не дает никакого приоритета, поэтому логичней размещать медиа-запросы в конце CSS документа, либо загружать тегом link внешний CSS файл с медиа-запросами после загрузки основных стилей сайта для правильного переопределения последних на разных разрешениях или на различных устройствах.

Для чего нужен CSS

На заре интернета сайты были максимально простыми и лаконичными. Многие из вас учились создавать их на основе HTML на уроках информатики. А потом появился CSS (аббревиатура от английского Cascading Style Sheets — каскадные таблицы стилей). Так у веб-страниц появилось оформление: отдельные блоки, шрифты, цвета и т.д. Современные сайты состоят из фронтенда — пользовательского интерфейса, и бэкенда — внутренней логики. Чтобы написать фронтенд разработчики используют три основных инструмента: HTML — язык разметки. Это своеобразный «чертеж» страницы. На HTML прописывают структуру: заголовки, ссылки, текст; CSS — код, отвечающий за визуальную часть страницы. Он нужен для того, чтобы на сайте были удобные кнопки, разные цвета и графика, а не только голый текст. Частично такие задачи можно решить и через HTML, но это будет очень долго и неудобно; JavaScript — язык программирования, на основе которого работают интерактивные элементы. Например, всплывающие формы, автозапуск видео или аудио, графика и т.д.
